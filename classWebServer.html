◆ tick()

int WebServer::tick ( )

Advances the web server's request processing by one clock cycle.

This method decrements the remaining processing time for the current request if the server is busy. Once the processing time reaches zero, the server is marked as idle, and a message is printed to indicate that the request is completed.

Returns
int The remaining processing time for the current request.
