RequestQueue::RequestQueue ( int totalRequests)

Constructs a new RequestQueue object and populates it with initial requests.

The constructor creates the request queue with a specified number of initial requests. Each request is generated with random IP addresses, a random processing time, and a random job type.

Parameters
totalRequestsThe number of initial requests to generate.
